# PhoneBook Frontend

React frontend for the PhoneBook app.

## Development

Start the backend first from the `backend` folder:

```sh
npm run dev
```

Then start the frontend from this folder:

```sh
npm run dev
```

The frontend calls `/api/persons`. During local development, Vite proxies `/api` requests to `http://localhost:3001`.

## Build

```sh
npm run build
```
