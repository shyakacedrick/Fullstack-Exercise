# Course Info App

This is a simple React application built as part of the Full Stack Open course.

## 📌 Purpose
The app displays information about a course, including:
- Course name
- Individual parts
- Number of exercises per part
- Total exercises

## 🧠 Concepts Learned
- React components
- Props (passing data between components)
- Component structure and reuse
- Basic JSX rendering

## 🚀 How to Run
```bash
npm install
npm run dev
